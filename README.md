### Un développeur Web Full-Stack et Web Designer.
Je créé des sites web, des APIs et des outils de gestion de serveur. Je code en **HTML**, **CSS**, **Js** (React.js, Node.js, Vue.js & Discord.js), **Shell**, **C++**, **C#**, **Ruby**, **Swift** et **Python**.
J'adore faire ça pour développer ma créativité, faciliter l'utilisation de certaines applications ou juste pour m'entraîner !

## 🔧 Mes projets 
- Une interface de **gestion d'autorité** de certification (avec mon alternance), disponible sur **[tools.zenetys.com/zpki](https://tools.zenetys.com/zpki/)**.
- Mon **[portfolio](https://sylvain.pro)**, un site rassemblant plein d'informations sur moi, mes **projets**, comment me contacter...
- Mon **[API personnelle](https://api.sylvain.pro)**, faite en Node.js, **hébergée** 24/7 !
- La **[documentation](https://docs.sylvain.pro)**, de mon **API** personnelle !
- Mon interface de **[logs](https://logs.sylvain.pro)**, pour l'**API**, pour voir les derniers **appels**...
- Mon outil de **[chats publics et privés](https://chat.sylvain.pro)**, vous pouvez y discuter en **public** dans un chat général **connecté** à mon **[site](https://sylvain.pro)** principal. Vous pouvez aussi **créer** vos propres chats **temporaires** privés.
- Le site où sont hébergés tous mes **[projets en PHP](https://php.sylvain.pro)**, avec toujours un moyen de les **tester** et **voir** ce que cela donne. Le **style** général est repris de mon **portfolio**.
- Mon **[terminal](https://terminal.sylvain.pro)**, réalisé avec **JavaScript** et qui contient les commandes de base d'un terminal **Linux**.
- **[Flowers](https://flowers.sylvain.pro)**, un LogCenter **customisable** en cours de **développement** !
- **[Digit](https://digit.sylvain.pro)**, une application web de reconnaissance de chiffres, basée sur un réseau de neurones en JavaScript.
- **[Readme](https://readme.sylvain.pro)**, une application web pour afficher vos fichiers **README** via une **URL** ou un **fichier**.
- **[Donut](https://donut.sylvain.pro)**, une **animation** d'un donut en rotation en **JavaScript** avec quelques **interactions** possibles.
- **[GemSync](https://gemsync.xyz)**, une plateforme pour vous aider à faciliter la **mise en ligne** de votre site Internet et la **configuration** d'un domaine personnalisé, et **bien plus** !
- **[GitSite](https://gitsite.dev)**, mon site d'hébergement, qui permet d'**héberger son site pour pas cher**, et qui prend en charge la configuration du DNS, des sous-domaines, etc.
